## 🔥 **ECONOMIST + COMPUTATIONAL LINGUISTICS**

✅ Regional Spanish Variants:
├── Colombia/MX: liability=pasivo, asset=activo
├── Spain: activo/pasivo (reversed order!)
├── Argentina: deuda emphasis
└── GAAP precision: accrual=devengo (MX) vs acumulación (ES)

✅ Key Differentiations:
liability (accounting)=pasivo ≠ liability (law)=responsabilidad
revenue=ingresos ≠ reventa (common Spanish error)

✅ 9-Column Features:
term,spanish,cefr,pos,ipa_noun,ipa_verb,region_notes,accountant_warning,economist_note
